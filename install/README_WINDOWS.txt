To install ShutDownPlus simply perform the following steps:

0. Extract "ShutDownPlus_v0_1_0_for_Windows.rar" with a tool like WinRAR or 7-Zip.
1. Run "install.bat".
2. Move the file "sdp.bat" to a place somewhere in your PATH variable.
3. If you want you can delete this whole extracted directory now. This is optional.

That's it.
ShutDownPlus can now be called from the command line via "sdp".
